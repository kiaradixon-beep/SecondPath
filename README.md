# Youth Behavior Tracking System

## Project Overview
The Youth Behavior Tracking System is designed to help educators and parents track and analyze the behaviors of young individuals. By gathering and analyzing data, we can better understand trends, patterns, and areas that may require attention or intervention.

## Features
- **Data Entry**: Allows users to input behaviors observed in youth.
- **Analysis Tools**: Provides visualizations and statistical analysis of behavior data.
- **Reporting**: Generates reports that can be shared with stakeholders like parents and educators.
- **User Management**: Supports different roles (e.g., educators, parents) with appropriate access to functionalities.
- **Notification System**: Alerts users about significant behavioral changes.

## Setup Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/{owner}/SecondPath.git
   cd SecondPath
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application**:
   ```bash
   python src/main.py
   ```

4. **Access the application via your web browser** at `http://localhost:5000`.
