# Youth Behavior Tracking System

A backend API for tracking youth behaviors, generating reports, and providing analytics for educators and parents.

## Features
- User Authentication (Register/Login)
- Behavior Logging
- Facility Management
- Analytics & Dashboard Endpoints
- Report Generation

## Setup

1. Install Node.js on your computer.
2. Clone this repository.
3. Run `npm install`.
4. Run `npm start`.
5. Open `http://localhost:3000` in your browser.
